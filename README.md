# 🏠 Airbnb Investment Advisor

**Airbnb Investment Advisor** is an interactive web application that helps users explore Airbnb market data and identify promising investment opportunities through data-driven insights and location-based analysis.

🌐 **Live Website:** https://airbnbinvestmentadvisor-xyz.streamlit.app/

## About the Project

This project was built to answer a simple question:

> **Where are the best Airbnb investment opportunities?**

Using Airbnb listing data, I performed data cleaning, exploratory data analysis, feature engineering, and SQL analysis to uncover patterns in pricing, location, and market performance. These insights were then transformed into an interactive web application where users can explore the data and discover investment-friendly areas.

## Features

- Interactive map-based property exploration
- Market and pricing insights
- Investment-focused metrics
- Location-based analysis
- User-friendly interface for exploring Airbnb opportunities
